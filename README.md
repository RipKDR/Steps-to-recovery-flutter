# Steps to Recovery (Flutter Rebuild)

Fresh rebuild of the app using Flutter + Dart.

## Stack
- Flutter 3.41.x
- Dart 3.11.x
- Material 3

## Included in v1
- Home dashboard with streak + next action
- Daily check-in (morning/evening)
- Journal page
- Progress page
- Support page
- Bottom navigation shell

## Run
```bash
flutter pub get

```
