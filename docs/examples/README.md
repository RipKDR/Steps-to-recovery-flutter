# Examples Directory

**These example files are for reference only and are NOT part of the main codebase.**

They demonstrate how to use the new packages but reference project-specific code that may not exist. Use them as inspiration for your own implementations.

## Files

| File | Package | Description |
|------|---------|-------------|
| `dio_client_example.dart` | Dio | HTTP client with interceptors |
| `lottie_examples.dart` | Lottie | Animation widgets |
| `haptic_feedback_examples.dart` | Haptic Feedback | Tactile feedback widgets |
| `mocktail_example_test.dart` | Mocktail | Modern mocking for tests |
| `golden_test_example.dart` | Golden Toolkit | Visual regression tests |
| `integration_test_example.dart` | Integration Test | E2E testing |

## How to Use

1. **Read the example** to understand the pattern
2. **Copy relevant code** to your actual implementation
3. **Adapt to your project structure** (update imports, class names)
4. **Test thoroughly** before committing
